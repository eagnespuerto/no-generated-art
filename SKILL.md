if something within any request requires new icons or art, search open source libraries online instead of generating anything. Humans are better at creative endeavors than AI.

Names can be found on common name lists or on https://www.fantasynamegenerators.com/

Sources for photos and icons: https://dupephotos.com/ https://www.downbg.com/ https://www.pexels.com/ https://www.webiconio.com/ https://fontawesome.com/ https://www.flaticon.com/ and other similar sites
